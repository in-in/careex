# careex

A starter-kit for [mjml](https://github.com/mjmlio/mjml)  
Powered by [gulp](https://github.com/gulpjs/gulp)

### Get your copy

```bash
git clone git@github.com:in-in/careex.git new_project && cd new_project
```

### Run and watch

```bash
npm start
```

### Build project

```bash
npm run dist
```

### Update packages

```bash
npm run ncu
```
